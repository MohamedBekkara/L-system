Ce fichier sert a vous expliquez comment lancé notre programme 
Suivez toute ces étapes si vous etes sur de correctement lancé le programme

- Tout d'abord créez ou reprennez un fichier avec le lsystem
definie a l'interieur ( dans notre cas le fichier entree)

- Ensuite ouvrez le terminal de votre ordinateur 
(si vous êtes sous Windows pouvez ouvrir powershell)

- Allez dans le repertoire ou ce situe ce programme avec 
la commande cd nomCompletRepertoire sur le terminale

- Assurez vous que python est correctement installer sur 
votre ordinateur avec la commande (python --version ou 
python3 --version ou py --version)

- Tapez la commande en remplacent l'entrée et la sortie par ce que vous voulez :
python3 main.py -i nomEntree -o nomSortie
ou
py main.py -i nomEntree -o nomSortie

A noter que vous n'êtes pas obliger de preciser la sortie, 
une sortie automatique se mettera si vous ne mettez rien

python3 main.py -i nomEntree
ou
py main.py -i nomEntree

- Lorsque le dessin du Lsystem sera fini cliquer juste 
sur l'ecran qui s'est ouvert pour quitter



Nous avons aussi placer des mini test unitaire dans 3 scripts
que vous pouvez appeler séparement :

lecture_fichier.py
substitution.py
creation_programme.py

en lancant les commandes : 
python3 lecture_fichier.py
python3 substitution.py
python3 creation_programme.py

Si vous voulez remplacer les tests unitaires de ces commandes 
par d'autres, allez à la fin de ces fichiers et modifié ce que 
vous voulez modifier, cela n'alternera la validité de tout le programme
